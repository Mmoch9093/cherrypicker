## 주요 특징
- WE:SH 시리즈: 일상 소비 전반을 커버하는 할인형 카드, 마이위시는 서비스팩 선택으로 맞춤 혜택 제공
- 톡톡 시리즈: 젊은 층 타겟의 실속형 카드, 낮은 전월실적에 높은 할인율
- 위시올: 전월실적 없이 국내 1% 할인 제공하는 범용 카드
- K패스 카드: 대중교통 10% 할인 + K패스 환급 이중 혜택
- 노리2 체크카드: 연회비 무료 체크카드로 커피/편의점 할인 제공
