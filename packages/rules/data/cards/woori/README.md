## 카드 파일

- `every1.yaml` - 무실적 전 가맹점 1% 할인
- `every-discount.yaml` - 무실적 0.8% 할인 + 간편결제 추가할인
- `every-point.yaml` - 무실적 0.8% 적립 + 간편결제 추가적립
- `shopping-plus.yaml` - 쇼핑/주유 특화 할인
- `point.yaml` - 기본 적립 + 간편결제 추가적립
- `point-plus.yaml` - 교통/통신 특별적립 + 간편결제 추가적립
- `card-of-rules-2.yaml` - 전 가맹점 1.2% 할인 (범용 최고 수준)
- `ten.yaml` - 일상 5대 영역 10% 할인
- `wowri.yaml` - 적립형 종합카드 (통신/교통/간편결제)
- `discount.yaml` - 전 가맹점 0.7~1% 할인 + 주유/교통/카페
- `da2.yaml` - 무실적 무한도 전 가맹점 0.8% + 생활업종 1.3%
- `oha-check.yaml` - 생활패턴 맞춤 체크카드
- `cookie-check.yaml` - 편의점/쇼핑/영화 체크카드
- `point-check.yaml` - 적립형 체크카드 (통신/백화점/마트)

## 참고

- 카드의정석 시리즈는 대부분 연회비 1.2만원으로 통일
- 카드의정석2는 연회비 2.2만원이나 전 가맹점 1.2% 할인으로 범용 카드 최고 수준
- EVERY 1은 무실적 무한도 1% 할인이 핵심 강점
- EVERY DISCOUNT는 기본 0.8% 무실적 할인에 간편결제 추가할인 가능
- EVERY POINT는 EVERY DISCOUNT의 적립형 버전
- DA카드의정석II는 무실적 무한도 0.8%+생활업종 1.3%로 DA@의 후속 상품
- SHOPPING+는 쇼핑 할인이 온라인/오프라인 구분되며 한도 별도 적용
- TEN은 커피/편의점/교통/통신/해외 5대 영역에서 10% 할인, 통합 월 최대 2만원
- WOWRI는 POINT+와 유사하나 주유 적립 추가, 간편결제 범위 넓음
- DISCOUNT는 전 가맹점 할인 + 주유/교통/카페 특화 (BC망 구형 상품)
- POINT 시리즈는 적립 시 결제건당 1만원 미만 제외
- 오하CHECK/COOKIE CHECK/POINT CHECK는 체크카드로 연회비 없음
- 공과금, 아파트관리비, 등록금 등은 대부분 적립/할인 제외
