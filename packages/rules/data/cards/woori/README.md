# 우리카드 (Woori Card)

> 마지막 업데이트: 2026-03-24

## 카드 목록

| 카드명 | 연회비 | 주요 혜택 | 전월실적 |
|--------|--------|-----------|----------|
| 카드의정석 EVERY 1 | 1.2만원 | 전 가맹점 무실적 1% 할인, 실적 시 최대 2% 할인 | 없음 (추가할인은 50만원~) |
| 카드의정석 EVERY DISCOUNT | 1.2만원 | 전 가맹점 0.8% 할인, 간편결제 추가 2% 할인 | 없음 (간편결제는 40만원~) |
| 카드의정석 SHOPPING+ | 국내 1만원/해외 1.2만원 | 온/오프라인 쇼핑 10%, 주유 60원/L, 카페 10% | 30만원 이상 |
| 카드의정석 POINT | 1.2만원 | 전 가맹점 0.8% 적립, 간편결제 3% 추가적립 | 없음 (특별적립은 30만원~) |
| 카드의정석 POINT+ | 1.2만원 | 전 가맹점 0.8% 적립, 교통/통신 5%, 간편결제 3% 추가적립 | 없음 (특별적립은 30만원~) |

## 카드 파일

- `every1.yaml` - 무실적 전 가맹점 1% 할인
- `every-discount.yaml` - 무실적 0.8% 할인 + 간편결제 추가할인
- `shopping-plus.yaml` - 쇼핑/주유 특화 할인
- `point.yaml` - 기본 적립 + 간편결제 추가적립
- `point-plus.yaml` - 교통/통신 특별적립 + 간편결제 추가적립

## 참고

- 카드의정석 시리즈는 대부분 연회비 1.2만원으로 통일
- EVERY 1은 무실적 무한도 1% 할인이 핵심 강점
- EVERY DISCOUNT는 기본 0.8% 무실적 할인에 간편결제 추가할인 가능
- SHOPPING+는 쇼핑 할인이 온라인/오프라인 구분되며 한도 별도 적용
- POINT 시리즈는 적립 시 결제건당 1만원 미만 제외
- 공과금, 아파트관리비, 등록금 등은 대부분 적립/할인 제외
