## 카드 파일

- `baro-clear-plus.yaml` - 연회비 5천원, 전월실적 15만원 진입장벽 최저 할인형
- `baro-reward-plus.yaml` - 기본적립 + 특별적립 결합 적립형
- `goat.yaml` - 무실적 국내 1.5%/해외 3% 적립 **(2026-03-06 발급 종료)**
- `baro-k-pass.yaml` - 대중교통 15% 할인 특화 교통 카드
- `baro-on-off.yaml` - 온/오프라인 + 해외 10% 균일 할인 범용
- `baro-payback-plus.yaml` - 무실적/무한도 0.7% 기본할인 보조카드
- `baro-quattro-plus.yaml` - 신세계/이마트/온라인몰 할인 특화
- `shinsegae-seven-flex.yaml` - 5개 카테고리 7% 할인 신세계 고객용
- `baro-macao.yaml` - 주유/장보기 변동형 할인 + 해외 2%
- `baro-air-plus-asiana.yaml` - 아시아나 마일리지 무실적 무한도 적립
- `shinsegae-fubilla.yaml` - 신세계그룹 무실적 무한도 할인 카드

## 특징

- BC카드(비씨카드)는 결제 프로세싱 네트워크이자 자체 카드 발행사입니다.
- 이 목록에는 BC카드가 직접 발행하는 "BC바로카드" 브랜드만 포함됩니다.
- 바로 클리어 플러스는 연회비 5천원, 전월실적 15만원으로 진입장벽이 가장 낮은 할인형 카드입니다.
- GOAT 바로카드는 전월실적 없이 국내 1.5%, 해외 3% 적립이 가능했으나 2026년 3월 6일 발급 종료되었습니다.
- 바로 리워드 플러스는 기본적립과 특별적립을 결합한 적립형 카드입니다.
- 바로 K-패스 카드는 대중교통 15% 할인에 특화된 교통 카드입니다.
- 바로 On&Off 카드는 온/오프라인과 해외까지 10% 균일 할인을 제공하는 범용 카드입니다.
- 바로 페이백 플러스는 무실적/무한도 0.7% 기본할인으로 보조카드로 적합합니다.
- 신세계 콰트로 플러스는 연회비 5천원으로 신세계/이마트/온라인몰 할인에 특화되었습니다.
- 신세계 세븐 플렉스는 5개 카테고리 전 영역 7% 할인으로 신세계 고객에게 최적입니다.
- 바로 마카오카드는 주유/장보기를 많이 할수록 할인율이 올라가는 변동형 할인 카드입니다.
- 바로 에어 플러스 아시아나는 무실적 무한도로 1천원당 1.3 아시아나 마일리지를 적립합니다.
- 신세계 푸빌라 카드는 신세계그룹 전체에서 무실적 무한도 할인을 제공하는 제휴 카드입니다.
