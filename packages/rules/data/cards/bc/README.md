# BC카드 (BC Card)

> 마지막 업데이트: 2026-03-24

## 카드 목록

| 카드명 | 연회비 | 주요 혜택 | 전월실적 |
|--------|--------|-----------|----------|
| BC 바로 클리어 플러스 | 5천원 | 점심/배달 7%, 온라인쇼핑 10%, 편의점 10%, 통신/스트리밍 10% | 15만원 이상 |
| BC 바로 리워드 플러스 | 1만원 | 전 가맹점 0.5~1.5% 기본적립, 영화/구독 30%, 쇼핑/주유 5% 특별적립 | 30만원 이상 |
| GOAT BC 바로카드 | 1.2만~1.5만원 | 무실적 국내 1.5%/해외 3% 페이북머니 적립 (한도 없음) | 없음 |
| BC 바로 K-패스 카드 | 6천~7천원 | 대중교통 15%, OTT/스트리밍 15%, 편의점 5%, 통신 5%, 해외 3% | 30만원 이상 |

## 특징

- BC카드(비씨카드)는 결제 프로세싱 네트워크이자 자체 카드 발행사입니다.
- 이 목록에는 BC카드가 직접 발행하는 "BC바로카드" 브랜드만 포함됩니다.
- 바로 클리어 플러스는 연회비 5천원, 전월실적 15만원으로 진입장벽이 가장 낮은 할인형 카드입니다.
- GOAT 바로카드는 전월실적 조건 없이 국내 1.5%, 해외 3% 적립이 가능한 무실적 카드입니다.
- 바로 리워드 플러스는 기본적립과 특별적립을 결합한 적립형 카드입니다.
- 바로 K-패스 카드는 대중교통 15% 할인에 특화된 교통 카드입니다.
