## 카드 파일

- `loca-365.yaml` - 생활고정비 할인 특화
- `loca-likit-1.2.yaml` - 무실적 전 가맹점 할인
- `loca-likit.yaml` - 스타벅스/영화/스트리밍 특화 할인
- `loca-likit-play.yaml` - 주유/영화/스트리밍 60% 할인
- `loca-100.yaml` - 마트/주유/통신 종합 할인
- `digiloca-london.yaml` - 무실적 캐시백
- `digiloca-paris.yaml` - 온라인쇼핑 + 기본할인
- `digiloca-monaco.yaml` - 프리미엄 무실적 할인
- `digiloca-lasvegas.yaml` - 해외 특화 무실적 할인
- `trip-to-loca.yaml` - 해외여행 특화 할인
- `dc-smart.yaml` - 생활비 종합 할인
- `loca-professional.yaml` - 프리미엄 무실적 캐시백
- `k-pass-loca.yaml` - K패스 대중교통 + 생활할인

## 참고

- LOCA 365는 건당 2만원 이상 결제 시 할인 적용
- LOCA LIKIT 1.2는 국세/지방세/공과금/대중교통/상품권 등 할인 제외
- LOCA LIKIT / LOCA LIKIT Play는 전월 40만원 이상 실적 필요, L.POINT 적립 미제공
- 디지로카 시리즈는 L.POINT 적립 미제공
- 디지로카 London은 즉시결제(선결제) 이용 시 추가 1.0% 캐시백
- 디지로카 Las Vegas는 무실적 해외 2% 할인으로 해외 여행 서브카드로 적합
- Trip to LOCA는 AMEX 브랜드 발급 시 공항라운지 무료 이용 가능
- LOCA Professional은 연회비 30만원이나 무실적 1% 무제한 캐시백으로 고소득/전문직 대상
- K패스엔로카는 대중교통 K패스 환급(일반 20%, 청년 30%) + 생활 할인 혜택
