## 카드 특징 요약

### 생활 할인형
Mr.Life는 공과금과 통신비 중심의 생활비 절약 카드이며, Discount Plan은 시간대별 할인이라는 독특한 구조를 가진 신규 출시 카드입니다.

### 교통 특화
B.Big은 대중교통 이용이 많은 고객에게, Deep Oil은 자차 운전자에게 최적화된 카드입니다.

### 적립형
ANNIVERSE(처음)는 OTT·구독 서비스 적립에 강점이 있으며, SOL Plan은 온라인쇼핑·배달 영역에서 높은 적립률을 제공합니다.

### 무실적/범용
Pli(플리)는 무연회비·무실적 카드로 서브카드나 첫 카드로 적합합니다.
