# IBK기업은행 (IBK Industrial Bank)

> 마지막 업데이트: 2026-03-24

## 카드 목록

| 카드명 | 연회비 | 주요 혜택 | 전월실적 |
|--------|--------|-----------|----------|
| IBK 일상의 기쁨카드 | 1만~1.2만원 | 커피 20%, 편의점/올리브영 10%, 대형마트 5%, 소셜커머스 20%, 주유 60원/L, 영화 1만원 할인 | 50만원 이상 |
| IBK K-패스카드 | 2천원 | 대중교통 건당 300원, 주유 40원/L, 통신 3%, 외식 5% | 20만원 이상 |
| IBK 하이브리드카드 | 무료 | 전 가맹점 신용 0.6% / 체크 0.3% 캐시백 | 없음 |
| IBK 마일앤조이카드 | 3만원 | 국내 1,500원당 1마일(대한항공), 해외 2~3배 적립, 공항라운지 | 30만원 이상 |

## 특징

- IBK기업은행은 BC카드 네트워크 기반으로 카드를 발행합니다.
- 일상의 기쁨카드는 생활밀착형 할인에 특화되어 IBK의 대표 카드입니다.
- K-패스카드는 낮은 연회비(2천원)와 낮은 전월실적(20만원)으로 대중교통 특화 카드입니다.
- 하이브리드카드는 무실적/무연회비로 신용과 체크를 하나로 결합한 카드입니다.
- 마일앤조이카드는 무실적 마일리지 적립이 가능한 여행 특화 카드입니다.
