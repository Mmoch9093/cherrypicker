- ONE 체크카드: 3종 캐시백(여기서더/모두다/369) 중 매월 선택 가능

  - 모두다: 오프라인 0.7%, 온라인 1.1~1.2% 무제한
  - 369: 1만원 이상 결제 3/6/9회마다 1000원
  - VIP: 전월 50만원 이상 시 오프라인/쿠팡 추가 2%
- MY 체크카드: 영역별 건당 300~500원 정액 캐시백 + K-패스 대중교통 환급 기능
- 모두의 카드: 대중교통 무제한 정액패스 (2026년 적용)
