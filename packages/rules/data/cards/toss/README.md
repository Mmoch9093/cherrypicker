|--------|--------|-----------|----------|

## 주요 특징
- 약관상 상시 혜택이 없는 프로모션 기반 카드 (스위치 캐시백 시즌 운영)
- 집중캐시백: 편의점, 대중교통, 택시, 카페&디저트, 패스트푸드, 영화관, 구독멤버십, 헬스&뷰티 8개 영역
- 어디서나캐시백: 국내 온/오프라인 0.3% 캐시백
- 해외결제 2% 무제한 캐시백 (공통)
- ATM 현금인출 수수료 면제 (공통)
- 캐시백 종류를 월 5회까지 변경 가능
