# NH농협카드 (NH Card)

> 마지막 업데이트: 2026-03-24

## 카드 목록

| 카드명 | 연회비 | 주요 혜택 | 전월실적 |
|--------|--------|-----------|----------|
| NH농협 올바른 FLEX 카드 | 1만~1.2만원 | 스타벅스 50%, 스트리밍 20%, 배달 10%, 대중교통 7%, 통신 7%, 영화 30% | 30만원 이상 |
| NH올원 파이카드 | 1만~1.2만원 | 온라인쇼핑 7~10%, 커피 20%, 배달 5%, 교통 5%, 주유/통신 선택 7~10% | 30만원 이상 |
| NH농협 올바른 POINT 카드 | 1.4만원 | 전 가맹점 0.7~1% 기본적립 + 쇼핑/편의점/영화/커피/해외 0.5~1% 추가적립 (한도 없음) | 30만원 이상 |
| NH농협 TAKE5(테이크5)카드 | 1만~1.2만원 | 5가지 팩 선택형: 외식 10%, 영화 30%, 서점 20%, 주유 60원/L 등 | 30만원 이상 |

## 특징

- NH농협카드는 농협은행을 기반으로 자체 카드 네트워크를 운영합니다.
- 올바른 FLEX 카드는 스타벅스 50% 할인이라는 파격 혜택으로 MZ세대에게 인기가 높습니다.
- 올원 파이카드는 최대 7개 영역에서 월 최대 5만원까지 혜택을 받을 수 있는 범용 카드입니다.
- 올바른 POINT 카드는 적립한도 없이 전 가맹점에서 포인트를 적립할 수 있는 것이 최대 장점입니다.
- TAKE5 카드는 매월 소비 패턴에 맞게 혜택 팩을 변경할 수 있는 맞춤형 카드입니다.
